# 评价记录

## 归档目录

考核文件归档在 `evaluation/` 目录下：

```
evaluation/
├── daily/              # 日常评价
├── self-summary/       # 各agent绩效总结
├── stage/              # 阶段考评
├── cycle/              # 周期复盘
├── personal/           # 个人考核
├── retrospective/      # 复盘报告
└── team/              # 团队变动
```
