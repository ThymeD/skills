# opencode-config-template

OpenCode 全局配置模板 - 提供全局 AGENTS.md 的通用配置参考，帮助新用户快速配置 OpenCode 环境。

## 功能

- 通用规则配置模板
- 推荐技能列表
- 使用说明
- 问题反馈流程
- 完整配置示例

## 使用

当用户需要创建 OpenCode 全局配置时说：
- "创建 OpenCode 全局配置"
- "初始化 OpenCode"
- "OpenCode 配置模板"

## 安装

```bash
# 克隆项目
git clone https://github.com/ThymeD/skills.git ~/.config/opencode/skills/thymed-skills
```

## 包含技能

| 技能 | 功能 | 安装方式 |
|------|------|----------|
| openclaw-ops | Gateway 运维 | 克隆项目 |
| skill-manager | 技能管理 | 克隆项目 |
| skill-finder | 技能推荐 | 克隆项目 |
| skill-creator | 创建技能 | 克隆项目 |
| find-skills | 发现技能 | `npx skills add find-skills -g` |
