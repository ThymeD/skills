# OpenCode 全局配置模板

提供全局 AGENTS.md 的通用配置参考，帮助新用户快速配置自己的 OpenCode 环境。

## 触发条件

- "创建 OpenCode 全局配置"
- "初始化 OpenCode"
- "OpenCode 配置模板"
- "参考我的 OpenCode 配置"

## 包含内容

1. **通用规则** - 语言规范、文件管理原则
2. **推荐 Skills** - 推荐的必备技能列表
3. **使用说明** - 技能发现和使用方法
4. **问题反馈** - 问题反馈与贡献指南

## 使用方式

将本技能提供的内容复制到你的全局 AGENTS.md 文件中：
- Windows: `%USERPROFILE%\.config\opencode\AGENTS.md`
- macOS/Linux: `~/.config/opencode/AGENTS.md`
