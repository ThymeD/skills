# Skill Creator

创建和优化 OpenCode skills。

## 功能

- 创建新 skill
- 优化现有 skill
- 测试验证 skill 效果

## 触发条件

- "创建一个 skill"
- "帮我写个 skill"
- "把这个变成 skill"

## 快速开始

1. 创建目录：`mkdir my-skill/`
2. 创建 `SKILL.md`
3. 可选：创建 `AGENTS.md`

## Skill 结构

```
skill-name/
├── SKILL.md (必需)
├── AGENTS.md (推荐)
├── README.md (可选)
├── scripts/ (可选)
└── assets/ (可选)
```
