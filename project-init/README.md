# Project Init

项目初始化。创建项目工作目录结构。

## 功能

- 检测项目工作目录是否存在
- 创建 OpenCode 工作目录（.poc/, requirements/, docs/, tests/）
- 引导用户开始开发

## 触发条件

- 开始开发
- 初始化项目
- 项目初始化
- 创建项目目录
- 开始做项目

## 目录结构

```
project/
├── .poc/                  # POC 验证目录
├── requirements/          # 需求清单
│   ├── SPEC.md           # 需求总览
│   └── features/         # 功能需求
├── docs/                 # 项目文档
└── tests/                # 测试文件
```

## 使用说明

1. 用户触发 skill 后，自动检测项目状态
2. 如果工作目录不存在，创建上述目录
3. 如果已存在，提示跳过
4. 输出创建结果并引导开发

## 重要说明

- 只在项目首次开发时触发一次
- 不会覆盖已有目录
