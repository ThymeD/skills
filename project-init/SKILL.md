# Project Init Skill

## 描述

创建项目工作目录结构，为项目开发提供工作目录。

## 触发词

当用户说以下话术时触发：
- 开始开发
- 初始化项目
- 项目初始化
- 创建项目目录
- 开始做项目

## 核心职责

### 1. 检测项目状态

- 检测项目是否已有工作目录
- 判断是否需要初始化

### 2. 创建目录结构

创建以下目录（如果不存在）：
```
project/
├── .poc/                  # POC 验证目录
├── requirements/          # 需求清单
│   ├── SPEC.md           # 需求总览
│   └── features/         # 功能需求
├── docs/                 # 项目文档
└── tests/                # 测试文件
```

### 3. 引导用户开始开发

输出：
1. 创建的目录结构
2. 后续使用说明

## 重要说明

- 只在项目首次开发时触发一次
- 不会覆盖已有目录
- 初始化是幂等的
