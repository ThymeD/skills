# Skill Sync PR

同步技能优化，将本地技能变更推送到 GitHub 或从远程更新。

## 功能

- 推送到远程：比较本地与 GitHub 仓库差异，用户确认后提交代码
- 从远程更新：检测远程最新内容，自动拉取并处理路径差异
- 代码审查：提交前确保无本地路径或敏感信息

## 触发条件

- "同步优化"
- "比较差异"
- "提交 PR"
- "从远程更新"
- "拉取最新技能"
- "检查技能更新"

## 使用方法

```bash
# 检查技能更新
从远程更新

# 同步本地修改到远程
同步优化
```

## 路径规范

所有路径使用跨平台写法：
- `~/.config/opencode/skills/` - OpenCode 全局技能目录
- `~/.config/opencode/skills/thymed-skills/` - GitHub 项目技能目录（git 仓库）
- `~/.config/opencode/skills/<其他>/` - 用户自定义技能
