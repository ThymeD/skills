# 处境分析逻辑

## 三种处境阶段的判断标准

### 失调期（Dissonance）
**特征**：对现状有强烈不满，但还没有方向感。
**信号**：
- Q1–Q4 答案情绪强烈、具体，有明确的怨恨或疲惫感
- Q5–Q7 描述让自己感到痛苦或恶心
- Q12–Q15 答案模糊、不确定，或者根本无法回答
- Q13 能清楚说出"敌人"，但不知道如何对抗

**典型表述**：「我知道我不想要什么，但不知道自己想要什么」

**建议基调**：你已经走了最难的第一步——看清了自己不想要的。现在需要的不是更多分析，而是小规模的实验和探索。

---

### 摸索迷失期（Uncertainty）
**特征**：有一定方向感，但行动力弱，容易被拉回旧模式。
**信号**：
- Q12–Q13 有答案，但措辞犹豫（"也许"、"应该"、"可能"）
- Q9–Q10 能识别出要放弃的身份，但感到代价很高
- Q11 想到了那件事，但没有真正去做
- 行动落地部分的答案每次回看都不一样

**典型表述**：「我知道方向，但总是做着做着就回到老样子」

**建议基调**：方向已经有了，现在需要的是降低行动门槛——不是做大事，而是每天做一件小到不可能失败的事。

---

### 清醒加速期（Discovery）
**特征**：有清晰愿景，行动一致，感到被某件事牵引。
**信号**：
- Q12–Q13 答案清晰、有画面感，情绪积极
- Q11 本周确实做了那件事
- Q14–Q15 两个句子写得有力量，不模糊
- 多次自检的答案之间有一致性和深化迹象
- 行动落地部分目标稳定，只是在细化

**典型表述**：「我有一个感觉，如果不做这件事，我会后悔」

**建议基调**：你已经找到了轨道。现在的任务是保护这种状态，减少能量泄漏，深化而不是拓宽。

---

## 核心模式识别

分析 Q1、Q2、Q9、Q10、Q11、Q13 的答案，寻找反复出现的主题词。

**常见模式类型**：

1. **认可驱动型**：答案中频繁出现"别人觉得"、"显得"、"被看见"、"不想让人失望"
   → 核心恐惧：失去他人认可
   → 建议：找到一件只有你自己知道的事情去做

2. **安全依赖型**：答案中频繁出现"稳定"、"风险"、"万一"、"退路"、"保险"
   → 核心恐惧：失控/失败带来的后果
   → 建议：做一个最坏情况分析，把恐惧具体化

3. **完美主义型**：答案中频繁出现"还没准备好"、"等条件成熟"、"不够好"、"做了也没用"
   → 核心恐惧：暴露自己的不足
   → 建议：把"完成"重新定义为"发出去"，而不是"做到最好"

4. **身份保护型**：答案中频繁出现"我一直都是"、"大家都知道我是"、"如果我改变了"
   → 核心恐惧：改变意味着之前的自己是错的
   → 建议：允许自己"一直在进化"，而不是"彻底改变"

5. **意义匮乏型**：答案普遍平淡，缺乏情绪强度，说不出自己想要或不想要什么
   → 核心状态：与自己的感受脱节
   → 建议：先不做计划，而是每天记录三件"让我感觉活着"的小事

## 分析输出格式

完成分析后，用以下格式输出：

```
━━━━━━━━━━━━━━━━━━━━━━
📍 当前处境：[失调期 / 摸索迷失期 / 清醒加速期]

🔍 你反复出现的模式：[模式类型 + 一句话说明]

⚡ 最大阻力来源：[基于 Q10/Q11/Q13 提炼，用用户自己的语言]

💬 一句话处境描述：
「[用第一人称，用用户答案中出现过的词，让他看到时觉得"对，就是这样"」

🎯 下一步建议：
1. [具体行动，今天或明天可以做，不超过15分钟]
2. [可选，下一周内可以做]

📅 建议下次自检时间：[具体到几天后，基于当前阶段]
━━━━━━━━━━━━━━━━━━━━━━
```

## 跨次记录对比逻辑

当用户有2次以上记录时，对比以下维度：

1. **情绪基调变化**：第一次 vs 最近一次，用更积极/消极/中性的措辞？
2. **愿景稳定性**：Q14 的答案每次都不同（探索中）还是越来越清晰（收敛中）？
3. **行动迹象**：Q11 的"那件事"，上次说了，这次有没有做？
4. **身份松动程度**：Q9 说要放弃的身份，有没有在答案中减少出现？

对比结论用一句话告知用户：
> 「对比上次，你对 X 的描述更清晰了，但在 Y 上仍然用同样的方式回避。」
