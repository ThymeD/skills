# Skill Finder

发现和推荐热门 AI Agent Skills。

## 功能

- 从 skills.sh 获取热门技能榜单
- 根据用户技术栈推荐合适技能
- 过滤已安装和不需要的类别

## 触发条件

- "有哪些热门 skills"
- "top skills"
- "推荐 skills"
- "有没有做 X 的 skill"

## 快速开始

```bash
# 查看热门技能
有哪些热门 skills？

# 个性化推荐
推荐适合我的 skills
```

## 偏好设置

偏好存储在 `~/.config/opencode/skills/skill-finder/preferences.json`
