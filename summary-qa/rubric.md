# 出题与内容规范（summary-qa）

## 目标

- **主目标**：帮助用户理解并掌握材料所表达的**关键知识点**，用于复习巩固。
- **非目标**：刁难、抠字眼、过度歧义辨析；不追求覆盖全文每一处细节。

## 出题数量

- **默认**：**5～10** 道题（单选 + 判断合计）。
- **可减少**：材料过短，或**可独立命题**的知识点不足（再凑则重复、牵强或偏离核心）时，可**少于 5 道**；须在步骤 1 或 Coverage 中**简要说明原因**。
- **用户指定（N ≤ 20）**：用户在本轮提示中**指定题数**且 **N ≤ 20** 时，**优先按用户意见**执行；材料**明显不足以支撑**时仍须说明或协商，不得无说明硬凑。
- **超过 20 题（强制）**：目标题数 **N > 20**（含显式数字或可合理推断的表述）时，**必须先与用户确认**后方可命题与输出题面；**禁止**未经确认输出任何题目。**确认后允许**按用户坚持的题数生成，且须为**每道题**标注质量等级（见下「题目质量等级」）。
- **补充**：若材料与 **N ≤ 20** 仍明显不匹配，须走 SKILL 中的**先报价再交货**（建议 **N'**、坚持 **N** 的后果、等用户拍板），**禁止**无协商直接硬凑。
- **先报价再交货（与 SKILL 一致）**：协商轮须包含建议题数 **N'** 及理由、若坚持原 **N** 的后果（重复考点、变体题、细枝末节增多、「低」质量题占比等），用户拍板后再出题。

## 题目质量等级（强制）

每道题须标注 **高**、**中**、**低** 之一；判定在命题时完成，**步骤 2 与步骤 3 同一题号须一致**。

### 质量判定的主要参考维度

命题时**优先**从下列维度衡量题目与材料的贴合度，再归入 **高 / 中 / 低**（三者是同一标尺上的相对档位，不是互斥标签）：

- **与主题、中心思想的契合度**：考查点是否落在**文章的主线、结论或反复出现的核心主张**上；越能代表「读完全文必须抓住的那条判断或结构」，越宜标 **高**。
- **作者希望读者重点理解的知识点**：标题、摘要、小节标题与首句、强调句、定义、转折、警示、方法步骤等**显式重点**；与**文眼、写作目的**呼应的段落。优先作为 **高** 或 **中**。
- **相对档位（与上两项一致）**：
  - **高**：不理解则难以把握**主旨或关键判断**；题干在卷内**无实质重复**；干扰项依托材料、非一眼假。
  - **中**：与主干相关但非**最顶层**的核心主张；或**明确但次要**的信息、对相邻内容的**合理综合**；仍可唯一定位出处。
  - **低**：**同考点变体**、近义复述、**与主旨距离较远**的边缘细节或辅证，或**信息增量有限**、易与其它题重复的表述；在大题量时用于**如实标注**「勉强可考」或「扩写凑数」类题目。**不得**将「低」标成「高」以粉饰质量。

**自检**：若某卷大量题为「低」，须在步骤 1 的 Coverage 或简述中说明原因（例如用户坚持超大 N）。

## 题目呈现顺序（强制）

- 向用户**展示与作答**时，须按质量等级 **高 → 中 → 低** 排列：**第 1 题**为卷内质量**最高**（最贴近主题/中心思想或最重点知识点），**第 N 题**为卷内质量**最低**。
- **命题流程**：先为每道题判定 **高/中/低**，再**按上述顺序排序**，最后赋予题号 **第 1～N 题**（题号与呈现顺序一致）；**禁止**在已排序卷面上随机打乱题号与质量的对应关系。
- **一次性题面**、**StudyPack 的 `questions` 数组**：顺序均须为 **高→中→低**（与步骤 2 一致）。
- **例外**：用户**明确要求**其它顺序（如按原文章节顺序、随机）时，**以用户为准**，并在步骤 1 或 Coverage 中**一句说明**已偏离默认顺序。

## 题型

- **仅允许**：单选题、判断题。
- **选用原则**：概念辨析、适用条件、因果关系等适合单选；是否与原文/作者观点一致、条件是否满足等适合判断题。**不强制比例**。
- **判断题的展示**：必须标出 **A、B** 两项（内容可为「正确 / 错误」或「是 / 否」等），与单选题的交互一致；不得只列文字而不带 A/B。

## 题干与选项

- 题干简洁；需要情境时，情境应**直接来源于材料**，不编造材料未支持的情节。
- 错误选项应体现**常见误读或忽略条件**，且与材料**强相关**；避免「一眼假」或与文章无关的干扰项。
- **不使用**「核对与争议点」中涉及的疑似错误、待核实内容来命题。

## 正确答案的位置（强制；避免选项偏置）

模型容易把正确答案**固定在同一字母**（例如总是 **B**），须主动打破这种偏置。

- **单选题（A–D）**：在确定题干与四个选项**内容**之后，将四个选项**随机打乱顺序**再标上 A、B、C、D，使正确答案落在 **A、B、C、D 中任意一项**；**禁止**在多道题中让正确答案**系统性**落在同一字母（若本卷题数 ≥4，理想情况下 A/B/C/D 均有机会出现；题数较少时至少**不要全部相同**）。
- **判断题（A/B）**：在确定「对 / 错」语义之后，**随机决定**「正确」与「错误」两项谁标为 **A**、谁标为 **B**（即有时「正确」在 A，有时在 B），**禁止**让多道判断题的正确项**总是**落在同一字母。
- **成卷后自检**：若发现**全部题目**的正确选项字母**完全相同**，必须**重新打乱**各题选项顺序后再输出（仅改顺序与字母标注，不改知识点与解析实质）。

## 解析

- 说明正确答案为何成立，并简要说明**易混淆点**（若存在），措辞避免钻牛角尖；混淆点须能**回扣到摘录出处**。

## 出处（默认）

- **章节标题（或小节标题）+ 摘录**；摘录长度以能**唯一定位**为准，通常一至三句。
- 无章节结构时：用**段落首句或自拟定位标题 + 摘录**。

## 语言

- **材料语种与输出语种分离**：材料为**英文或其它外文**时，总结、题干、选项、解析仍须默认**简体中文**，**不得**整卷改用英文；详见 [SKILL.md](SKILL.md)「语言规范」之**输出语种（强制）**。
- 总结、题干、选项、解析默认**简体中文**。
- 代码、命令、专有名词、出处摘录等按 [SKILL.md](SKILL.md) 中的语言规范处理。

## 展示与交互（对用户）

- **题目顺序**：默认按 **质量从高到低** 呈现（第 1 题最重要），见上文「题目呈现顺序」；用户另有要求时除外。
- **答题前**（步骤 1–2）：只展示题面（题干 + 选项）及每题 **质量：高 / 中 / 低**，**不得**同时给出正确答案或整卷解析。
- **题面标题行（推荐）**：`第 N 题（质量：高｜中｜低）｜单选题` 或 `｜判断题`，下一行空行再接题干；选项单选为 **A.～D.** 每项一行。详见 [SKILL.md](SKILL.md) 步骤 2「题面排版（推荐）」与 [examples.md](examples.md)「推荐题面排版」。
- **默认逐题**：步骤 1 后应直接出第 1 题，**不要**让用户在逐题/一次性之间二选一；一次性题面仅当用户在本轮调用时已明确要求（见 SKILL.md）。
- **默认推荐与用户意图**：无相反说明时遵循 [SKILL.md](SKILL.md)「宿主环境与会话形态」中的**默认推荐**；用户**明确指定**的流程（只要总结、一次性题面、同条合并输出等）**优先**，细则与例外见该节。
- **答题后**（步骤 3 起）：先**总体结果表**，再**逐题明细表**（列顺序见 SKILL：题号、题型、质量、结果、完整题干、你的选择、正确答案），最后**逐题解析**列表；**文章全文式总结**放在步骤 4（见 SKILL.md）。
- **输入解析**：`1–4` 对应单选 `A–D`；判断题 `1、2` 对应 `A、B`；字母选项忽略大小写；无法唯一解读时须二次确认后再记题（见 SKILL.md）。

## 自检清单（生成后内部核对）

- [ ] 出题数量符合 SKILL / 上文「出题数量」（默认 5～10；不足可减少；**N > 20 须已确认**；材料不匹配须已说明）。
- [ ] 每题均有 **质量：高 / 中 / 低**，且步骤 2 与步骤 3 一致。
- [ ] 题目已按 **高 → 中 → 低** 排序并编号为第 1～N 题（用户要求其它顺序时已说明例外）。
- [ ] 每题均有明确出处（标题 + 摘录）。
- [ ] 单选题仅有一个最佳答案；判断题无歧义到需猜度。
- [ ] 未用「题后争议内容」充当考点。
- [ ] 材料过长时已输出 Coverage 覆盖声明（如适用）。
- [ ] 对用户输出顺序符合 SKILL：完成通知 → 逐题作答 → 判分讲评 → 文章总结 → 明显问题提醒（可选）。
- [ ] 步骤 2 题面符合 SKILL「题面排版（推荐）」或等价信息完整（题号、质量、题型、题干、选项分行）。
- [ ] **用户可见输出**为**简体中文**（材料为外文时亦然；用户明确要求其它语言时除外）。
- [ ] 正确答案在 A/B/C/D（及判断题 A/B）上**分布随机或均衡**，无「全是 B」等系统性偏置。
